# Eksamensprojekt | Gruppe 2:
## Projektkalkulationsværktøj

Et webbaseret værktøj til at nedbryde projekter, beregne tidsforbrug og visualisere deadlines for Alpha Solutions.

## Om projektet


## Teknologier
* Backend: Java 21, Spring Boot 3.5.6
* Frontend: Thymeleaf, HTML, CSS
* Database: MySQL 8.0
* Build Tool: Maven
* CI/CD: GitHub Actions
* Hosting: Azure Web App + Azure MySQL
* IDE: IntelliJ IDEA 2025.2

## Installation

1. Klon repository:

   ```git clone https://github.com/KEA-BarHjaMadMik/Eksamensprojekt.git```

2. Importér projektet i IntelliJ IDEA.
3. Konfigurer database:
   * Opret MySQL database.
      * [TO-DO]
   * Tilføj url og credentials som miljø-variabler:
      * DEV_DATABASE_URL
      * DEV_USERNAME
      * DEV_PASSWORD
